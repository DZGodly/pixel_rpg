# Changelog

## v0.9.0 - 2026-04-13
**提示词**: 大型内容扩展 - 主线任务、Boss战、技能树、新区域

**改动**:
- 6阶段主线任务系统：城市管理员→工厂Boss→地下通道→AI核心→量子霸主→通关
- 5个支线任务：零件收集、数据采样、佣兵委托、失踪工人、黑市通行证
- 3个Boss敌人：失控监工(工厂)、觉醒AI核心(数据中心)、量子霸主·真身(网络空间)
- 2个新区域：地下通道(35x25，连接工厂↔数据中心)、黑市(25x20，隐藏商店区)
- 12节点技能树，3分支(攻击/防御/黑客)各4层，升级获得技能点解锁
- 8个新物品：精密齿轮、数据样本、加密数据、工人证件、等离子步枪、纳米装甲、黑客手套、解毒程序
- 5个新普通敌人：管道蠕虫、安保无人机、纳米虫群、防火墙守卫、数据吞噬者
- NPC对话根据主线进度动态变化(quest_dialogues)
- Boss触发点固定在地图特定位置，需满足任务阶段才触发
- 区域门控：地下通道需击败工厂Boss，黑市需完成通行证支线
- 通关动画+自由探索模式
- 黑市军火商出售独占高级装备
- 所有新数据支持存档/读档
- 2种新地图图块：管道地板、锈蚀墙

## v0.8.0 - 2026-04-13
**提示词**: 房子内部平铺整个屏幕，丰富内容，增加NPC和关联对话

**改动**:
- 室内地图从 10x8 扩大到 30x20，铺满整个屏幕
- 6个房间各有独特布局：维修工坊(隔墙+工作台)、数据分析室(终端阵列)、退休黑客书房(书架环绕)、酒吧(吧台+卡座+舞池)、改装店(展示柜+等候区)、线人密室(多隔间)
- 新增4种室内装饰图块：书架、沙发、地毯、吧台
- 室内NPC从6个增加到14个，每个房间2~3人
- NPC对话形成关联情报网：维修技师↔工厂工人、分析师→退休黑客、酒保传递分析师消息、改装师↔线人互相推荐、逃亡工人串联工厂线索
- 室内宝箱从6个增加到11个
- 门口传送点改为双格宽

## v0.5.0 - 2026-04-11
**提示词**: 把代码模块化拆分，避免 context 爆掉

**改动**:
- 原 2251 行 game.py 拆分为 8 个模块文件
- constants.py (70行) - 常量、颜色、工具函数
- assets.py (345行) - 程序化素材生成
- particles.py (59行) - 粒子系统
- game_map.py (151行) - 地图数据与生成
- entities.py (190行) - 物品、玩家、NPC、敌人定义
- combat.py (319行) - 战斗系统
- dialogue.py (224行) - 对话系统
- game.py (914行) - 主游戏类与入口
