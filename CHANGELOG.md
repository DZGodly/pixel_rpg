# Changelog

## v0.3.0 - 2026-04-11
**提示词**: 优化 button 图标，让商人能出售商品

**改动**:
- NPC 交互提示从 `[Z]` 统一为 `[J]`，加气泡背景样式
- 幽灵商人交互提示同样加气泡背景
- 商店系统新增出售功能，左右键切换「购买/出售」tab
- 出售价格为购买价的一半，无购买价的物品按 15G 基准
- 购买时显示是否买得起（✗ 标记）
- 商店操作增加反馈消息（购买成功/金币不足/卖出获得）

## v0.2.0 - 2026-04-11
**提示词**: 普通敌人的遇敌概率下调，增加击杀敌人的奖励

**改动**:
- 遇敌概率从 0.8% 降至 0.4%
- 史莱姆奖励 10exp/5g → 18exp/10g
- 蝙蝠奖励 12exp/8g → 20exp/15g
- 骷髅战士奖励 25exp/15g → 45exp/30g
- 暗影龙奖励 100exp/80g → 180exp/150g
- 战斗掉落概率 30% → 50%

## v0.1.0 - 2026-04-11
**提示词**: 添加随机隐藏元素，增加重玩价值和探索乐趣

**改动**:
- 新增隐藏宝箱系统（每区域随机 1-2 个，按 J 发现）
- 新增金色史莱姆稀有敌人（1% 概率遭遇，高经验高金币）
- 新增随机事件（捡金币/发现药水/踩陷阱/神秘祝福）
- 新增幽灵商人 NPC（森林中闪烁出现，卖稀有物品）
- 新增物品：万能药水（全恢复）、幸运金币（atk+3）
- 隐藏宝箱和幽灵商人状态纳入存档系统
